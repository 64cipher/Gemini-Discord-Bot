# Gemini-Discord-Bot
simple bot avec Gemini

```pip install discord```

```pip install google.generativeai```

```pip install asyncio```

Configurez le bot en mettant l'API Gemini, le token bot et l'ID du salon de votre serveur

```python GemBot.py```
